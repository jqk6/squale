Avertissement

Avant de procéder à l'installation de Squale, vérifiez que votre système possède bien la configuration minimale suivante :

  • présence d'un JDK, dont la version est supérieure à 1.5
  • la variable d'environnement JAVA_HOME pointe vers le répertoire d'installation de ce JDK

Merci aussi de ne pas installer Squale dans un répertoire dont le chemin contient des espaces.
