Welcome to Software QUALity Enhancement!

Squale is an open-source project which aims at assessing software quality, with the following requirements :

  • ability to analyze different languages, using third party technologies (may they be open-source or commercial, depending on the platform),
  • use of enhanced quality models to aggregate raw quality data into high level factors,
  • reporting adapted to every profile, ranging from technical details for developers to consolidated dashboards for top-level managers,
  • action plan proposals dedicated to developers, which give them clues and tips on the best way to improve faster their code quality,
  • estimation of the return of investment of quality efforts, based on models that use economical indicators and correlate them with quality data.
